Dummy Text
API

API Documentation

The Dummy Text API has 1 resource.

https://dummytext.co/api

Parameters

There are two optional parameters that you can include with your requests:

  • category - The category of dummy text to be generated. Defaults to 'mixed' which uses a potentially different category for each sentence in a paragraph.
  • paragraphs - The number of paragraphs between 1 and 20. Defaults to 1.

An example request:

fetch('https://dummytext.co/api?category=tiktok&paragraphs=3')
    .then(response => response.json())
    .then(data => console.log(data))
